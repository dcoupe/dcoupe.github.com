写代码最重要的还是区分0o,li1这些，字体再好看点就更好了

目前我用的是Monaco感觉很不错哦

[下载地址](http://www.gringod.com/wp-upload/MONACO.TTF )
[来源](http://www.gringod.com/2006/02/24/return-of-monacottf/)

安装

```bash
cp MONACO.ttf /usr/share/fonts/truetype/
fc-cache -f -v
```
